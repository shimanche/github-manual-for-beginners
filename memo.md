# git initで初期化
$ git init
ローカルのリポジトリが作成されます

# commit コミットの仕方
vscodeのターミナルで、
git config --global user.email "you@example.com"
git config --global user.name "Your Name"
を入力する

$ git commit -m "comment "

## error
[no changes added to commit]
* 最初にステージする
* 次にコミットする
* use "git add" and/or "git commit -a"

# push　pushの仕方
$  git push origin main
